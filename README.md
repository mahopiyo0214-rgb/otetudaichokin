# おてつだいアプリ

タブレットでぽちぽち押して、毎月のお手伝い回数と合計金額を出すシンプルなアプリです。
小学生でも直感的に使えるよう、大きなボタン・絵文字・ひらがな多めで作っています。

## つかいかた

1. `index.html` をブラウザで開くだけで動きます (インストール不要)
2. お手伝いをしたら、そのカードを **ぽちっと押す** だけでカウントされます
3. 画面右上に **今月の合計回数と合計金額** が表示されます
4. まちがえて押した時は、カードの左上の `−` ボタンで 1 回分けせます

データはブラウザ内 (`localStorage`) に保存されるので、同じ端末・同じブラウザで開けば前の記録が残っています。

## 一般公開のURL（おすすめ：GitHub に 1 回 push するだけ）

このフォルダには **GitHub Actions** の設定が入っています。push すると **gh-pages というブランチ**にサイト用ファイルだけが自動で載り替わり、**どのパソコン・タブレットでも開ける https の URL** で公開できます。

### 手順（はじめてのときだけ）

1. [GitHub](https://github.com) で **新しい空のリポジトリ**をつくる（名前は例: `otetsudai-chokin`）。**Public** にする。
2. このフォルダでターミナルを開き、次を実行する（`<あなたのユーザー名>` と `<リポジトリ名>` を置き換え）。

```bash
cd "/Users/wmba2026/Desktop/AIディレクターコース/shiori/お手伝いアプリ"
git init
git branch -M main
git add .
git commit -m "お手伝い貯金アプリ"
git remote add origin https://github.com/<あなたのユーザー名>/<リポジトリ名>.git
git push -u origin main
```

3. **Actions** タブを開き、**Deploy to GitHub Pages** が **緑のチェック**で終わるまで待つ（1〜2分）。失敗したら **Re-run jobs** を一度試す。
4. GitHub のリポジトリ → **Settings（設定）** → **Pages（ページ）** を開く。
5. **Build and deployment** の **Source（ソース）** で **Deploy from a branch（ブランチからデプロイ）** を選ぶ（**GitHub Actions ではない**方です）。
6. **Branch** で **`gh-pages`** を選び、右のフォルダは **`/ (root)`** のまま。**Save** があれば保存。

※ 以前 **GitHub Actions** をソースにしていた場合は、必ず **ブランチから** に切り替えてください（今のワークフローはその方式です）。

7. 同じ **Pages** の画面に、**サイトの URL**（**Visit site** など）が出ます。

公開URLの形はだいたい次のとおりです（リポジトリ名で変わります）。

`https://<あなたのユーザー名>.github.io/<リポジトリ名>/`

この **https の URL** を LINE やメールで送れば、**別のパソコンやタブレットのブラウザ**からそのまま開けます。

### あとからアプリを直したとき

同じフォルダで変更を保存したあと、

```bash
git add .
git commit -m "更新内容のメモ"
git push
```

だけで、また自動でサイトが更新されます。

### ほかの方法（ドラッグだけで URL）

- [Netlify Drop](https://app.netlify.com/drop) に、このフォルダごとドロップする方法もあります（GitHub を使わないとき向け）。

### 注意（データ）

記録は**ブラウザと端末ごと**に別です。同じURLを共有しても、**別のパソコンでは記録はつながりません**。タブレットとパソコンで同じ数字を見たい場合は、今の版では同じ端末で開くか、将来「書き出し」機能が必要です。

### おなじ家の Wi‑Fi だけで試す

`npx --yes serve .` で出るローカル URL は、同じ家の中の機器から試すとき向けです（PC を切ると止まります）。

## ボタンの説明

| ボタン | はたらき |
|---|---|
| 📅 りれき | 月ごとの合計とお手伝いの内訳を見られる |
| ⚙️ せってい | おうちの人用。項目の名前・金額・絵文字を変えたり、追加・削除ができる |

## 月の切り替わり

月が変わると、トップに表示される今月分の集計が自動でリセットされ、前の月のものは「りれき」から見られます。

## へいじつ・休日 の きんがく

**ぽちっと押したときの日付（日本時間）**で決まります。

- **へいじつ** … そのこうもくの **へいじつ（円）**
- **休日** … そのこうもくの **休日（円）**

「休日」の日に入るのは次のとおりです。

- **土曜・日曜**
- **日本の祝日**（振替休日・国民の休日もふくむ。`japanese-holidays` ライブラリで判定）

カードには **へいじつ** と **休日** の金額を、オレンジの行で 2 段に表示します。合計・一覧・りれきには、押した日に応じた片方の金額が入ります。トーストには「（へいじつ）」「（休日）」または「（しゅくじつ）」と出ます。

## 入っているお手伝い (デフォルト)

休日の金額をきほんに、へいじつは だいたい ÷0.9 を四捨五入した値でそろえています（せっていで自由に変えられます）。

| 絵文字 | おてつだい | 休日 | へいじつ |
|---|---|---:|---:|
| 🍽️ | おさらあらい | ¥100 | ¥111 |
| 👕 | せんたくものをかたづける | ¥50 | ¥56 |
| 🌀 | かんそうきにいれる | ¥20 | ¥22 |
| 🛁 | おふろそうじ | ¥150 | ¥167 |
| 👟 | くつをそろえる | ¥10 | ¥11 |
| 🐱 | ペットのおせわ | ¥100 | ¥111 |
| 🍳 | おりょうりのおてつだい | ¥50 | ¥56 |

ごみだし・へやのそうじ・おかいものは いまの デフォルトから はずしています。項目は「せってい」から ふやしたり けしたりできます。

## タブレットで使うコツ

- ホーム画面に追加すると全画面で使えます
  - iPad (Safari): 共有ボタン → 「ホーム画面に追加」
  - Android (Chrome): メニュー → 「ホーム画面に追加」
- 画面回転は横向き推奨 (縦でも使えます)
